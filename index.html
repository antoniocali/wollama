How this works

This project is a simple JSON-backed directory of Ollama models that people have successfully run on their local hardware. GitHub Pages hosts the site; contributions are made via pull requests editing models.json.

Detected hardware is best-effort: the browser exposes only limited information. You can always override using the search bar to filter by GPU/CPU, RAM, or device name.

Want to contribute? Add or tweak entries in models.json describing: the model, purpose, whether tools are supported, and the hardware it runs well on. Then add yourself to contributors.json so we can thank you publicly.

Contributors

This project is community-driven. Everyone who contributes models or improvements is listed here. Add yourself by editing contributors.json in a PR.
